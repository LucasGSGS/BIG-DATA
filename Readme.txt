This folder contains output files for analysis done on big occupancy matrix. 
For non-negative matrix factorization Rank = 20, beta = 0, number of cluster = 15.

" Analysis on Normalized data"


cluster_name : Contains names of parking lot present in each cluster.
clusters: contains all the parking lots in each cluster. 
corelation matrix: contains corelation matrix for each cluster.
signatures: contains signatures for each cluster
visualizations, visualizations1 : contains visualizations of parking lots and and signatures of each cluster.
